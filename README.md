# QA Engineer Assessment - SauceDemo Testing

## Overview
This project contains manual testing, API testing, and automation testing for SauceDemo and JSONPlaceholder APIs.

---

## Tools Used
- Playwright
- JavaScript
- Postman / Console API testing
- GitHub Codespaces

---

## Project Structure

manual-testing/
api-testing/
automation/

---

## How to Run Automation Tests

1. Install dependencies:
npm install

2. Install Playwright browsers:
npx playwright install

3. Run tests:
npx playwright test automation/

---

## Manual Testing
Contains test cases and bug reports for SauceDemo.

---

## API Testing
Uses JSONPlaceholder API:
- GET /users
- POST /posts
- PUT /posts
- Edge cases included
